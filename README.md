# Restaurant DB GUI

## Основные возможности

- Управление ресторанами и их меню
- Резервация столов
- Обработка и отслеживание заказов
- Учёт ингредиентов и автоматическое формирование заявок поставщикам
- Ролевой доступ (официант, повар, менеджер, администратор)
- Генерация отчётов по производительности сотрудников
- Поддержка полнотекстового поиска (с учётом русского языка)
- Защита от SQL-инъекций и управление безопасностью через RLS (Row-Level Security)
